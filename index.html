System Philosophy

Portfolio-centric, role-guarded, and intentionally quiet until real work appears.

Guilds define learning domains. Curators create modules. Seekers build portfolios and niche folders. Arbiters review platform activity without editing it. No dashboard contains seeded profiles, modules, or portfolio work.

Verification

Curator permissions require a valid Curator Code. Arbiter authority is manually approved in code only.

Storage
